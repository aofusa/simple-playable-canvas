Simple Playable Canvas
=====


Canvas要素に画像の表示とキーボードで操作するのサンプル


環境構築
-----
```sh
npm install
```

実行
-----
```sh
npm run dev
```

http://localhost:5173 にアクセス  
以下の画面が表示されるのでカーソルキーかWASDで操作できる
